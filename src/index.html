PurinWare
This game is a work in progress. It is not finished yet. But it is playable.


Story

Purin is the boss of PurinWare Inc, you are a new employee tasked with helping Purin going through her day.

Context

This game is inspired from the WarioWare Inc game, but with Purin instead of Wario.
When Purin played WarioWare inc on stream, I thought she may like a game like this with her in it.
It use a mix of redraw version of the original game asset and completely new asset.
The code is completely written from zero.


Part 1 - Morning


Stop Her - LVL 1-3

Get Purin Attention 3 time to wake her up
Control the machine with the 'Z' key

Iron It - LVL 4

Iron Purin Clothes
Move the iron with the Arrow keys

Dress Up - LVL 5-8

Purin Need help dressing up
Move Purin with the Arrow Left/Right keys

Shake It - LVL 9

Get her breakfast ready
Shake the flan with the Arrow Left/Right key

Part 2 - Afternoon


Stop Her - LVL 10-13

Get Purin Attention 3 time to get her ready for her lunch
Control the machine with the 'Z' key

Note: The game is not finished yet, the level after 13 are not playable yet.
